# Telegram Selling Bot

Production-ready Telegram bot for selling digital products with UPI payment support.

The application source lives in [bot/](./bot), including setup instructions in [bot/README.md](./bot/README.md).

## Quick Start

```bash
git clone https://github.com/beast-69-bot/telegram-selling-bot.git
cd telegram-selling-bot/bot
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
copy .env.example .env
python main.py
```
